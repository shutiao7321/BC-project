MOTIVATION

展覽動機與目的

這不是一場告別，而是一次暫停。我們想在喧囂的蟬鳴響起前，為這四年的瑣碎、熱血與安靜找一個棲身之處。 透過數位空間的轉化，將那些快要模糊的記憶重新對焦。本展覽旨在建立一個跨越時間的橋樑，讓即將啟程的我們， 能在未來的任何時刻，回頭看見此刻最明亮、最純粹的自己。

INTRODUCTION

展覽介紹

本展覽以「紀事」為名，紀錄從 2022 年入學到 2026 年離校的點滴。 展區分為四大核心：由 GitHub API 驅動的動態影像清單、基於雲端同步的實時交流牆、 可客製化並導出的數位明信片工作坊，以及釋放壓力的趣味互動裝置。 我們誠邀您穿梭於影像與文字之間，感受這段集體記憶的重量。
